# DogTinder
Developed a DogTinder using Bootstrap Components
## About The Project
This project is a Bootstrap-based landing page inspired by the TinDog project from App Brewery.
It was recreated and customized as part of my learning process in responsive web design and Bootstrap layout systems.

## Original reference project:
TinDog by App Brewery – https://appbrewery.github.io/tindog/

## Built With
*HTML5
*CSS3
*Bootstrap 5
